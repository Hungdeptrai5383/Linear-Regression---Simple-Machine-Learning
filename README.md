# Linear-Regression---Simple-Machine-Learning
This project implements Linear Regression using pure Python, from scratch

## Features
- Manual gradient descent
- Mean Squared Error loss
- Fully object-oriented structure

## Project Structure
model.py      → Model definition
calculate_error.py     → Loss function
train.py      → Training logic
main.py       → Program entry point

## How It Works
The model learns parameters m and b in the form:

y = mx + b

Using gradient descent to minimize MSE loss.

## How To Run
python main.py

## Example Output
Trained m ≈ 2
Trained b ≈ 0
